### Key Patterns Discovered

- Domain entities use `Entity<T>` base with mutable `props: T` object (NOT frozen — unlike ValueObject)
- `Result<T, E>` is used everywhere — no exceptions for expected failures
- `reconstitute(id, props)` static method on each entity for loading from DB (no domain events fired)
- `create(...)` static factory fires domain events
- Supabase repos use `upsert` for both insert and update
- Application layer use cases inject repos + eventBus interfaces (no infrastructure imports)
- WebSocket broadcaster `EnrichedEventBus` is used for actual WS broadcast in routes
- The container.ts (`server/src/api/container.ts`) wires everything together
